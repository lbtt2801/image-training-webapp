# AI Image Training WebApp

## Cài đặt & Chạy

1. Tạo venv: python -m venv venv
2. Activate: source venv/bin/activate (Linux/Mac) hoặc venv\Scripts\activate (Windows)
3. Cài: pip install -r requirements.txt
4. Chạy: cd backend && uvicorn main:app --reload
5. Mở: http://127.0.0.1:8000
